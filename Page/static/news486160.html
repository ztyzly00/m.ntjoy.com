西安17岁女生遭宿舍5女孩欺凌患抑郁 见人发抖

2016-05-20 14:47:13 江海明珠网
0

　　近日，17岁女生小宜看到陌生人靠近，就会浑身发抖，蜷缩起来说“别打我”，而她这样的反应，只是因为连续两次遭受同宿舍5名女孩的群殴。5月19日上午11时，在长安区西安市精神卫生中心，医生对小宜进行精神检查后，在病历上写着：意识清，表情呆滞，问话少答，提起事情经过全身震颤、哭泣，诊断为轻度抑郁，建议住院进一步诊治。图为5月19日上午，在西安市精神卫生中心，家人和华商报记者安慰受惊吓的小宜（化名 ) 。

　　到底发生了什么事情，让小宜如此恐惧和不安？小宜的母亲说，他们是蓝田人，小宜在长安区的西安工程技师学院上学，学的是学前教育。5月14日下午6时许，她突然接到小宜班主任的电话，让她到学校去一趟，说孩子出了点事情。赶到学校后他们才从老师那里知道，小宜被同宿舍的5名女孩不止一次的群殴，当日下午4时许，宿舍的一名女孩让小宜给她去买饭，小宜说学校餐厅没卖的，那名女孩气冲冲地说，如果她去了发现有卖的，不会放过小宜。小宜害怕极了，发短信向其他同学求救，随后班主任老师就赶到了小宜的宿舍。他们了解到这个情况后，立即报了警。图为受惊吓的小宜（化名 ) 。

　　从小宜的手机上还可以看到当日的求救短信，5月14日下午4时11分，她跟同学发送短信，“你马上让班主任来我宿舍，要不我就没命了。”小宜的姑父说，之前小宜从来没跟家人说过在学校被打的事情，因为这次的事情他们一再追问，才知道小宜之前也被打过，但是被威胁不许告诉老师和家人，否则会被打得更惨。同宿舍的几名女孩经常让小宜帮她们买饭、洗袜子，若有不从或反抗，就会被打，而且每次都是在晚上宿舍熄灯以后。图为小宜家人在宿舍收拾孩子物品。

　　“孩子跟我说，她甚至被打得装死，我听得心都碎了。”小宜的母亲哭着说。看到母亲哭得伤心，小宜低着头说“我害怕”，在反复的开导后，小宜才说，宿舍的女孩用拖鞋扇她的脸，她看见拖鞋就害怕。小宜的母亲提供了事发后5月15日带小宜到省医院的就诊病历，诊断证明上写着“殴打致伤后全身多处疼痛”，“胸外伤，胸壁软组织挫伤”，“受群打数日，受打后闭经”。图为西安工程技师学院长安校区，设有专门的心理咨询室。

　　5月19日中午1时许，在小宜学校的保卫处，一名负责调查此事的郭老师说，14日下午，他接到学校电话就赶紧从家赶到学校，并连夜将此事涉及的所有当事人叫到保卫处，对事情经过进行了详细的了解。目前，学校对程某、邓某某、刘某某做出开除学籍、留校察看处理，对张某、刘某某给予记过处分。
