歼15飞行员张超牺牲细节公开：全程只有4.4秒

2016-07-31 14:46:44 江海明珠网
0

　　航空母舰被称为“国之重器”，是强大海军必不可少的组成部分。2013年5月，中国海军组建了第一支舰载航空兵部队，向打造航母战斗力迈出了关键步伐。今年4月27日，海军某舰载航空兵部队一级飞行员张超在执行训练任务时，不幸壮烈牺牲。他是为中国航母舰载机事业殉职的第一位飞行员。

　　盛夏时节，渤海湾畔的葫芦岛某飞行训练基地，骄阳刺眼，海风呼啸。海军舰载战斗机飞行员们正在加紧训练。

　　这批飞行员都已完成了数百小时的飞行科目训练。按照计划，他们即将迎来服役至今最重要的一飞——第一次赴航母“辽宁舰”进行着舰、起飞训练。

　　作为其中一员，飞行员袁伟的心中交织着复杂的感情：“他在即将完成上舰梦想的时候，发生了这个事情，挺可惜的。我们都很难受。上舰的时候，我将带着自己的梦想，也会把他的梦想一同带上！”

　　袁伟口中的“他”，是好战友和好兄弟，海军一级飞行员张超。三个多月前，在脚下的这片训练场上，一场事故突如其来——张超驾驶歼15战机进行陆基模拟着舰接地时，因飞机突遇故障，壮烈牺牲，年仅29岁。他成为了为中国航母舰载机事业献身的第一位英烈。

　　从视频监控记录上看，在生命最后的4.4秒，张超为尽最大可能保住战机，奋力拉下制动杆，阻止机头上扬，却因此错过了最佳跳伞时机，坠地后身负重伤。

　　在海军某舰载机战斗团参谋长徐英眼中，张超的行动展现了崇高的责任和使命感：“飞行员对飞机的感情，一般人是理解不了的。飞行员驾驶战鹰，两者就像兄弟一样。在遇到这种故障的时候，飞行员的第一反应肯定是挽救飞机，挽救国家财产。这正是他那一天所做的壮举。”

　　大国海军，离不开航空母舰。2012年，随着“辽宁舰”入列服役，海内外高度关注中国航母建设的一举一动。次年5月，中国海军组建了第一支舰载航空兵部队，向打造航母战斗力迈出了重要一步。

　　2014年下半年，得知海军选拔舰载机飞行员的消息后，张超婉拒原部队的挽留，申请加入这支使命神圣的部队。那时候，他已经是服役11年的优秀三代机飞行员，先后驾驶过6种机型。

　　让海军某舰载航空兵部队部队长戴明盟至今难忘的是，张超在面试时，表现出了少有的坚定信念： “我问他，‘你知不知道我们部队是什么样的一支部队？跟以前的飞行有不一样的地方？’他说，‘知道’。我问他，‘你知不知道这里的风险这么大？’他说，‘也知道’。我又问他，‘你还愿意来吗？’他说，‘愿意，就是喜欢飞行’。”

　　航母战斗机飞行员，因极高的风险，被喻为“刀尖舞者”——一份国际报告显示，舰载战斗机飞行员的风险系数是航天员的5倍，普通飞行员的20倍。西方某大国发展航母初始阶段，平均每两天摔1架飞机，牺牲了1000多名飞行员。目前，全世界现役的舰载机飞行员不过2000人。

　　军事专家指出，由于航母着舰使用反区技术，与传统战机截然不同，即便是优秀的三代机飞行员，跨过“这道坎”，通常需要一年半到两年的努力。

　　在战友的记忆中，张超是个脸上总挂着笑容的大男孩儿，对待业务精益求精。海军某舰载机战斗团参谋长徐英告诉记者，每逢训练间隙或者讲评阶段，张超总是跟教练、战友们探讨飞行细节得失，他还发挥自身优势，参与战机作战教材的编写：“有一次，我们针对一个问题，一天没有讨论完，吃完晚饭，他又到我宿舍，讨论到熄灯前才把这个问题搞明白。他来部队前，飞的是另一个机型，有驾机实战经验，我会利用他的经验，对目前这款飞机开发它的作战潜能。我一次交给他一个任务，他会加班加点，做了很多工作。”作为特招的“插班生”，张超仅仅花了半年时间，就完成了同班学员近一年的训练课程。他的飞行成绩是班上的“佼佼者”，已经达到上舰标准。如果不是这次突发事故，他即将驾驶战鹰，完成上舰首飞的夙愿。

　　在妻子张亚眼中，张超视飞行事业为生命。从相识到结伴的六年里，小两口团聚的时间只有一年多。张超没给妻子过一次生日，补拍婚纱照的承诺最终没能兑现。而今，2岁多的女儿每每思念爸爸，就哭喊着翻看手机里的照片。

　　“人生有酒当醉否，一滴可曾到九泉”，在朋友圈中，张亚这样抒写对丈夫的无尽思念。在她内心，丈夫的人生是完美的：“站在妻子的角度，确实很难接受（这个现实）。但人的一生并不能用生命的长短去衡量好坏。我认为，他的一生非常完美，唯独留下的遗憾，是没有等到（上舰）的那一刻。他的战友会替他完成这个事业。他会感到欣慰的！”如果把中国的航母事业比作一场“万里长征”，眼下，才刚刚起步。只有尽早打造出一支强大的舰载机飞行员队伍，建立起先进、完备的人员培训体系，中国航母才能真正形成战斗力。张超和他的战友们，用生命和热血，成为这一伟业的先驱。

　　“航母战斗机英雄试飞员”、海军某舰载航空兵部队部队长戴明盟认为，白手起家发展航母事业，前方可预知和不可预知的风险和挑战还有很多，但为了让祖国的航母早日驰骋深蓝，捍卫国家的海空安全，他们将矢志不渝，砥砺前行：“我们现在仅仅是在渤海飞行。未来，我们要走向大洋。今后还有更长的路要走。我们一定会秉承张超同志的遗志，把后面的工作做好，而不是说沉浸在悲痛中，止步不前。”
