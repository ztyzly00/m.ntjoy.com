伊能静剖腹伤口感染化脓 斥美医生失医德

2016-08-08 09:36:46 江海明珠网
0

　　47岁伊能静今年6月为老公秦昊，在美国剖腹产下一女小米粒，却因伤口感染，8月5日再度入院开刀清创；她6日除发文感谢大家对她的关心，同时也控诉为她接生的医生“没医德”，不仅害她伤口感染，该医院更未经她同意，就将她照片用为宣传广告。

伊能静

　　伊能静赴美生产并不顺利，她今日发文，指控该医院“真的很有问题”，未经她同意就将她与医生的合照到处宣传广告，质问医生照片为何外流？医生却回覆“医院里有人偷他手机，将照片发出去，才把手机还给他”。

　　接着伊能静更揭开她剖腹伤口感染严重原因，表示在伤口线裂开时，医生强调贴美容胶就会好，还塞了一罐抗生素给她，过了一周胶带就全被化脓侵蚀，发出异味，但该医生依旧强调伤口没问题、只是裂开。

　　事后伊能静疼痛难耐，找了另一位友人推荐医生求诊，才得知伤口感染太严重，还在验血时，发现先前医生开的抗生素，对她完全无效。原来当初伊能静裂开的伤口已有感染，贴美容胶的作法只是让化脓细菌全往刀口裡流，让‘小伤口变成整个内部全被侵蚀感染’。

　　即便如此，伊能静依旧保持正面能量，在文末表示“上天选中我，正是因为我有影响力，能让更多人知道找医生一定要多问，多了解医生的医德与医术”，也强调关于医生手机被偷、导致病人隐私外流一事，她会保留法律追诉权，呼吁该医院别再拿她当宣传，去伤害更多妈妈。
