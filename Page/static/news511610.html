春秋航空禁止员工带三星Note 7手机登机

2016-09-14 15:59:58 江海明珠网
0

受三星 Galaxy Note7手机爆炸及航司接连抵制的冲击，三星股价连日来跌跌不休。

　　又有一家航空公司加入“封杀” 三星Galaxy Note 7的行列。

　　9月14日，春秋航空发布安全预警，再次重申旅客应严格遵守中国民航局安全规定，飞行全程中，严禁使用包括三星Galaxy Note 7在内的任何手机和充电，或作为托运行李运输。

　　春秋航空在安全预警中表示，近期三星公司Galaxy Note 7上市仅一个多月,在全球范围内已经发生了35起因锂电池质量缺陷造成起火爆炸的事故。目前美国 FAA、 欧洲 EASA、日本国土交通省等国家/地区民航当局已经将Note7列入危险品之列，禁止旅客携带Note 7登机。同时国内个别航空公司已经禁止旅客在飞机上使用三星Note7手机或充电。

　　春秋航空做出风险预警包括：

　　1，春秋航空任何航班上不接受旅客使用三星Galaxy Note7手机和充电，或作为托运行李运输；

　　2，公司所有干部员工乘坐本公司航班时，禁止携带三星Galaxy Note7手机登机，同时禁止将其作为托运行李运输；

　　3，值机时应主动询问旅客，并做好告知工作；

　　4，起飞前广播中特别增加禁止旅客使用三星Galaxy Note 7手机或充电的内容;在飞行过程中加强客舱监控，一旦发现旅客使用手机或给手机充电，立即制止，随时做好应急情况准备和处置。

　　春秋航空并非第一家“封杀” 三星Galaxy Note 7手机的中国航空公司。

　　9月13日，海航集团安全管理委员会下发的通知显示，海航禁止集团所有干部员工携带Galaxy Note 7设备登机，且禁止将其随行李托运；海航集团旗下航空运输企业不接受旅客在飞机上使用Galaxy Note 7、充电或随行李托运；加强中国民航局关于严谨在飞行中使用手机等相关电子产品规定的宣传。

　　海航方面进一步向澎湃新闻表示，海航根据民航局相关安全管理规定，下发内部操作风险防范通知，进一步加强对旅客携带和使用电子产品的监控管理，加大巡查力度，及时制止旅客在飞行期间使用手机或为手机充电，严禁含锂电池手机随行李托运等影响飞行安全的行为，后续还将通过网站、值机柜台等相关渠道下发安全提示，严格管控安全风险。

　　相较于春秋航空的规定，海航的规定更具有“杀伤力”。除了海南航空，海航集团运营管理的天津航空、金鹿公务、祥鹏航空、首都航空、西部航空、福州航空、乌鲁木齐航空、北部湾航空、扬子江航空、桂林航空、长安航空、myCARGO、加纳AWA航空、法国蓝鹰航空等航空公司，也得统一执行海航集团的三星Galaxy Note7手机“封杀令”。

　　日本国土交通省9月9日发布类似警告，要求国内各航空公司不要让乘客在飞机上使用三星Galaxy Note 7手机。

　　韩联社报道，韩国交通部9月10日发表声明，要求三星Galaxy Note 7手机用户在乘坐飞机时全程关机、不给手机充电，且手机不能放在托运行李内。

　　此前，美国联邦航空局发表声明，“强烈建议”乘客在登机后不要启动三星Galaxy Note7手机或为其充电，也不要把它放在托运行李里面。新加坡航空公司、澳大利亚航空公司、维珍澳大利亚航空公司和阿联酋阿提哈德航空等公司也纷纷加入这一行列，为三星Galaxy Note7手机下达飞行禁令。

　　三星Galaxy Note7手机发布一个多月以来，已在全球范围内发生三十多起因电池缺陷造成的爆炸和起火事故。韩国三星公司不得不宣布，因电池缺陷问题，将在10个国家和地区停售三星 Galaxy Note 7手机，并允许已经购买到这款手机的消费者两周后置换新手机。据估算，召回事件将让三星的营收减少50亿美元，年净利润降低5%。

　　受三星 Galaxy Note7手机爆炸及航司接连抵制的冲击，三星股价连日来跌跌不休，9月12日开盘闪崩，截止收盘跌幅已接近7%，创下自2012年以来最大跌幅，单天市值蒸发超过100亿美元。

　　截至发稿，中国民航局尚未就三星 Galaxy Note7手机的使用作出统一规定。
