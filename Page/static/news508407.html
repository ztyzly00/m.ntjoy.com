樊馨蔓：愿照顾张纪中私生子 或分上亿财产

2016-08-31 09:36:32 江海明珠网
0

　　8月30日，樊馨蔓临时召开简短的记者会，并再次向新浪娱乐说明了自己与张纪中的婚姻状况，还指出张纪中根本没有才华，自己还为他当过枪手，并透露看过张的私生子照片，如张纪中今后无法照顾，自己愿意代为照顾。据樊馨蔓保守估计，她至少可以拿到上亿财产。

　　当天早些时候，某媒体曝樊馨蔓出轨干儿子肖齐，得知此消息后，樊馨蔓接受了新浪娱乐的独家专访，不仅澄清了自己与肖齐只是工作关系，还爆料真正出轨、转移财产的人是张纪中，他更有小31岁的女友以及3岁的私生子。

　　此后，樊馨蔓在微博上表态：“这年头，做个正常的正派人，应该不难吧？”由于求证的媒体过多，她把记者们都请到了所谓的“绯闻地”纪中文化公司，她表示主要是“一个个地说很累，再一个呢，人有直觉，你看着我就能知道我是什么样的人。”

　　樊馨蔓曝张纪中初中文化水平 自己曾任枪手

　　张纪中是业内知名制片人，但樊馨蔓却爆料称，张纪中本是北京的下乡知青，连初中都没毕业，文化水平低下的他却一直不肯放弃文艺美梦。樊馨蔓多次帮助他，她透露，张纪中当年之所以选中赖以成名的金庸剧，就是樊馨蔓和马云共同确定的，樊还为张纪中写书并心甘情愿地署名张纪中。

　　樊馨蔓说，自己算是张纪中制片人路上的启蒙老师，为其购买好莱坞制片人手册，解答邵兵[微博]版令狐冲不得力怎么办的大问题，花费20余年打造出中国第一制片人的形象。

　　樊馨蔓表示她没有想到，这些努力没有拉近两个人的距离，她越来越发现，张纪中并没有因此改变。她透露，马云赠送给张纪中华谊股票后，导致了张纪中发生了质的变化。张纪中的日常变成了每天在家里躺着看着股票笑，樊馨蔓形容他和神经病一样。

　　此后，樊馨蔓认为张纪中的思想已经停滞不前，并开始大肆购买奢侈品，并主动要求加入美国籍。樊馨蔓告诉新浪娱乐，她在与张纪中的电话中，“问过他那些事情，他认为有美国法律保护他。”

　　肖齐曾被张纪中视作棋子 却与樊馨蔓交好

　　2006年，樊馨蔓和张纪中都是认识了肖齐的。2009、2010年，樊馨蔓已经找不到张纪中，两人的婚姻就已经亮起红灯，当时包括肖齐在内的朋友们都知道，此后两人已经形同陌路。

　　此后，张纪中多次受挫，单独拍摄的《英雄时代》至今没发。2012年，肖齐被张纪中引入纪中文化，据樊馨蔓自己分析，张纪中一直在金钱控制上面无法与樊馨蔓对抗，因此找来肖齐掣肘。而肖齐本人，是直到2013年，才知道樊馨蔓跟张纪中的关系已经破裂。

　　后来，肖齐与樊馨蔓的合作次数增多，建立起了工作上的信任，樊馨蔓还答应肖齐，纪中文化的作品，都由他任监制，更是提出要让肖齐做总裁，但却惨遭张纪中反对，“张纪中是很愤怒的，因为他认为自己是最牛的。”

　　樊馨蔓斥张纪中相当不地道，仅仅参与了与电影《战城蚩尤》无关的拍摄，后期剪辑都是樊馨蔓团队所完成的，至今仍未上映，就是因为樊不答应将他挂为电影总导演。

　　樊馨蔓自曝至少分得一亿 愿照顾张纪中私生子

　　谈起张纪中的外遇，樊馨蔓表示，她发现张纪中与小三就住在她与张之前的房子里。目前，张纪中的私人物品、牙刷、衣物都保存在家里，等着张纪中来取走。

　　至于为什么到今天才把这些事公之于众，樊馨蔓说，以前没有想到这样做能得到什么，只是不想做和张纪中抢钱的女人。

　　不过，在财产分割中，樊馨蔓保守估计自己能拿到上亿的财产。据了解，纪中文化的法人是樊馨蔓，张纪中为股东。

　　作为母亲，樊馨蔓希望自己能够拿到孩子的抚养权。另外，她透露自己早已看过张纪中私生子的照片了，孩子十分可爱，她表示如果张纪中以后因为什么事不能照顾孩子，她会照顾这个孩子。（阿辉/文）
